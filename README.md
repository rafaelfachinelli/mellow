## :arrow_forward: **Run**

### :desktop_computer: **WEB Responsive**

<details>
  <summary><i>with <b>npm</b></i></summary>
  
  ```bash
  # Install dependencies
  $ npm install

  # Start development server
  $ npm start
  ```
  
</details>

<details>
  <summary><i>with <b>yarn</b></i></summary>
  
  ```bash
  # Install dependencies
  $ yarn

  # Start development server
  $ yarn start

  ```

</details>
